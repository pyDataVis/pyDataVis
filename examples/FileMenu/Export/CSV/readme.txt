Test of the option "Export to CSV" in the File menu

Open the file "ExportCSV.plt"

Select the option "Export to CSV" in the File menu.

Input a file name in the file selector which pops up.

If you choose the suggested name (ExportCSV.csv), this generate two CSV files (one for each data block) named as ExportCSV-1.csv and ExportCSV-2.csv.
