Test of the option "Import from spreadsheet" in the File menu

With a LibreOffice calc file:
- Open the file "import-1Sheet.ods" which contains one data sheet.
- Use the option "Import from spreadsheet"
pyDataVis has created one data block to store the data.


With an Excel file:
- Open the file "import-2Sheet.xls" which contains two data sheet.
- Use the option "Import from spreadsheet"
pyDataVis has created two data blocks to store the data from each sheet.
