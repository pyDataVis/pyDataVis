Test for "Transpose" option in Convert menu.

The "Transpose" option switches the rows and columns of a data block.

Open the file "Transpose.txt". It contains 9 vectors with only 2 elements. The number of elements is to small for plotting the curves.

Select the "Transpose" option in Convert menu.

This create a new file using the same name as the initial ".txt" file but with ".plt" extension (in this case "Transpose.plt"). This file is loaded in a new *pyDataVis* window. It contains two vectors with 9 elements.
