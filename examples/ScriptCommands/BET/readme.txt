Test for "BET" script command.

The first data block must contains the adsorption isotherm with the first vector of this block being the relative pressure and the second vector the adsorbed volume expressed in mL/g.

Open the file "AdsDes-iso.plt".

Type "BET "in the Script window and execute (Ctrl+E).

This open an information window giving the optimum BET surface area, the BET constant and the pore polume (see BET.png image).
